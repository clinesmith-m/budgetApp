SEND SPENDING CATEGORIES:
INTERFACE -> GCAT
BACKEND   -> <NUM CATEGORIES (4 bytes)>
for each category...
    BACKEND -> <CATEGORY NAME LEN (4bytes)>
    BACKEND -> <CATEGORY NAME>
    BACKEND -> <AMOUNT BUDGETED (double)>
    BACKEND -> <AMOUNT UNSPENT (double)>
INTERFACE -> T

MODIFY SPENDING CATEGORY:
INTERFACE   -> MCAT
INTERFACE   -> <CATEGORY NAME>
INTERFACE   -> <AMOUNT BUDGETED (DOUBLE)>
BACKEND     -> T

ADD SPENDING CATEGORY:
INTERFACE   -> ACAT
INTERFACE   -> <CATEGORY NAME>
INTERFACE   -> <AMOUNT BUDGETED (DOUBLE)>
BACKEND     -> T

ADD MONTHLY EXPENSE:
INTERFACE   -> AMNE
INTERFACE   -> <MEMO>
INTERFACE   -> <AMOUNT (DOUBLE)>
INTERFACE   -> <NUM MONTHS (SHORT)>
BACKEND     -> T

CANCEL MONTHLY EXPENSE:
INTERFACE   -> CMNE
INTERFACE   -> <MEMO>
BACKEND     -> T

GET MONTHLY EXPENSES:
INTERFACE   -> GMNE
BACKEND     -> <NUM MONTHLY EXPS (INT)>
For each expense...
    BACKEND     -> <MEMO>
    BACKEND     -> <AMOUNT (DOUBLE)>
INTERFACE   -> T

    BACKEND     -> <MEMO>
